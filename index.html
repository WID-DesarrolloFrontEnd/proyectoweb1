HyperText Markup Language

fundamentos de HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Se usa para estructurar el contenido de una página mediante etiquetas que definen elementos como textos, imágenes, enlaces, inputs, div, navbar, footer, forms, tablas y formularios.

Cascading Style Sheets

fundamentos de CSS

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para definir la apariencia y el diseño de una página web escrita en HTML. Con CSS, puedes cambiar colores, fuentes, márgenes, tamaños, posiciones y mucho más, haciendo que los sitios web sean más atractivos y funcionales.
